## General Info

A simple chat app built with Websockets, React and Redux.

## Technologies 

Project is created with:
* React
* Redux
* Websockets
* SASS
* CSS Modules
* Webpack
* Babel
* ES6

## Set up

1. Run `npm install`
2. Start the dev server using `npm start`
3. Open [http://localhost:3000](http://localhost:3000)
